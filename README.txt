== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `aiaibot.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
2. Set the correct *Integration Code* in aiaibot chatbot settings form.
